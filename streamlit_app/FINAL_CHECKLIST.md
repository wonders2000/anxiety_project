## 📞 Quick Troubleshooting

| Problem | Solution |
|---------|----------|
| "module not found" | `pip install -r requirements.txt` |
| "webcam not found" | Check permissions/device |
| "model not loading" | Verify json + h5 files exist |
| "port in use" | `streamlit run streamlit_app.py --server.port 8502` |
| "slow performance" | Reduce FPS slider in app |

More help: See `STREAMLIT_README.md` Troubleshooting section
